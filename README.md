# Bayesian Regression Analysis in R

This project investigates factors associated with fertility rates across 19th-century Swiss regions using Bayesian linear regression in R.

The analysis was completed using R Markdown and demonstrates Bayesian statistical modelling, posterior inference, model comparison, diagnostic checking, and predictive performance evaluation.

---

## Overview

The project explores relationships between fertility and several demographic variables from the Swiss dataset, including:

- Examination scores
- Catholic population percentage
- Agriculture
- Education
- Infant mortality

Bayesian linear regression with conjugate priors is used throughout the analysis to estimate posterior distributions and quantify parameter uncertainty.

---

## Methods

The analysis includes:

- Exploratory data analysis
- Bayesian linear regression
- Posterior parameter estimation
- Posterior predictive visualisation
- Bayes factor model comparison
- Variable transformation
- Model diagnostics
- Train-test validation using RMSE

The final model uses a log-transformed response together with selected explanatory variables to improve predictive performance and model assumptions.

---

## Key Results

The project demonstrates:

- Posterior estimation of regression coefficients
- Posterior sampling and uncertainty visualisation
- Comparison of competing Bayesian models using Bayes factors
- Residual diagnostics
- Predictive performance evaluation using RMSE
- Final model selection based on predictive accuracy and interpretability
